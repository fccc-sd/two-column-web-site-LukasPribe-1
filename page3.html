Game Description

This is the first game I've made it is very basic and doesn't have much to it. The game is pong with one player controlling the left paddle with the W and S keys and another player controlling the right paddle with the up and down arrow keys. The ball moves very slow when compared to the paddles so it it is hard for one player to score unless the other player makes a mistake or allows them to score.
